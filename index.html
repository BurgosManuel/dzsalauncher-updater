# How it works?

- I made this simple website to request the server list from the official DZSA Launcher API, then i parse it into a downloadable "dayz-servers.json" file.

- This is useful to get rid of the "There was a problem downloading server list one or more errors occurred" error.

# How to use?

- Wait until the server list is retrieved, and after that, simply click the button below to start the download

- The server name and IP:PORT will also be displayed below so you can check your favourite servers information.

# How to fix the error?

- Copy and paste the "dayz-servers.json" into Documents/dzsalauncher/
